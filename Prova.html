Respostas

1) Durante o acesso a uma página na web, ao pesquisar um site e confirmar, o navegador utilizado pelo usuário utiliza o DNS pra transformar o nome do domínio em um endereço IP, o que gera uma conexão TCP/IP com o servidor, enviando uma solicitação HTTP/HTTPS. Desse modo, o servidor te responde com os dados do site, renderizando eles para exibição pro usuário.


2) A frase está incorreta, pois o HTML não é uma linguagem de programação, mas sim uma linguagem de marcação! Por exemplo, no HTML não tem variáveis iguais outras linguagens de programação como JavaScript e Java por exemplo. E no HTML é usado tags.


3) As regras no CSS funcionam através da tag "style", na qual fica armazenado todo o conteúdo do CSS utilizado. No CSS há diversas regras, como por exemplo, para você acessar uma classe no CSS, utiliza-se um "." antes do nome da classe informada, por exemplo ".container{}". E para o CSS funcionar, você tem que abrir chaves e fechar as chaves, se não, não irá funcionar. A mesma coisa serve para o ";" ao final de cada linha informada no CSS, como por exemplo, "border: 1px solid black;", se não utilizar o ponto e vírgula ao final, não funcionará. Temos também os "id", que para acessá-los no CSS, utiliza-se "#" + nome do id, por exemplo, "#imagem{}", acessando a tag do html que tem esse id, podendo utilizar o CSS para estilizá-lo.
